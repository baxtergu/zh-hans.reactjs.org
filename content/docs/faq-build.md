### 我怎样才能在 JSX 中写注释？ {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* 注释写在这里 */}
  Hello, {name}!
</div>
```

```jsx
<div>
  {/* 多行注释 
  也同样有效。 */}
  Hello, {name}! 
</div>
```
